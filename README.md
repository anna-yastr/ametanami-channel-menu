# Бот-меню канала — шпаргалка

Разделы: `stickers` `comics` `art` `personal` `games`

---

## Обложки разделов (photos.json)

Отправить фото/гиф/mp4 боту с подписью:

| Подпись | Действие |
|---|---|
| `main` | заменить главную обложку |
| `stickers` | заменить обложку раздела |
| `stickers +` | добавить в галерею обложек |
| `stickers 1` | заменить картинки **поста №1** |
| `stickers 1 +` | добавить к картинкам **поста №1** |

Подпись ставится на первое фото альбома — работает и для пачки.

**Очистить обложки раздела:**
```
/clear_stickers
/clear_comics
```

## Посты в подразделах (posts.json)

### Создать пост

**Карусель картинок** (картинки добавляются отдельно через фото):
```
/add_post comics Внутри
/add_post art Ведьмочки
```

**Скопировать из приватного канала** (контент появляется в чате нативно):
```
/add_copy stickers 4472109036 3 Пак стикеров
/add_copy comics 4472109036 4,5,6,7,8 Внутри
```
Формат: `/add_copy раздел айди_канала номера_через_запятую название`
Номер поста: правая кнопка на сообщение в канале → «Копировать ссылку» → последнее число в ссылке

**Ссылка на публичный пост** (кнопка открывает Telegram):
```
/add_link comics https://t.me/имя_канала/42 Внутри
```

### Посмотреть список постов раздела
```
/list_posts comics
```

### Удалить/очистить

| Команда | Действие |
|---|---|
| `/remove_post comics 1` | удалить пост №1 целиком |
| `/clear_post comics 1` | очистить картинки поста №1 (пост остаётся) |

## Подписи под отдельными картинками

По умолчанию под картинкой показывается название поста/раздела.
Чтобы добавить свой текст под конкретной картинкой — добавь поле `"caption"` в её запись в **posts.json** или **photos.json**:

```json
{
  "file_id": "AgAC...",
  "type": "photo",
  "caption": "Сатаплия — древний лес с динозаврами"
}
```

Чтобы картинка была вообще без подписи:
```json
{"file_id": "AgAC...", "type": "photo", "caption": ""}
```

Работает везде: обложки разделов, посты, подпункты, галереи.
После изменения файла — перезапустить бота.

## Прямое редактирование posts.json

Структура поста-карусели:
```json
{"title": "Внутри", "images": []}
```

Структура поста из приватного канала:
```json
{"title": "Внутри", "channel": "4472109036", "ids": [4, 5, 6]}
```

Структура поста-ссылки (публичный канал):
```json
{"title": "Внутри", "url": "https://t.me/имя_канала/42"}
```

После редактирования файла вручную — перезапустить бота.

```
py -3 main.py
```
